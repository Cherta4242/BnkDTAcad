<h3>문제 3</h3>
- Servlet을 저장소(Repository) 역할로 사용하여 학생의 성적을 처리하는 웹 프로그램을 작성하시오.
1. 프로젝트 생성
- 새로운 Dynamic Web Project를 생성한다.
- 프로젝트 이름은 Domain으로 한다.
2. Student 클래스 작성
- domain 패키지에 Student 클래스를 작성한다.
- 학생의 이름, 국어, 영어, 수학 점수를 저장할 수 있도록 한다.
3. 입력 페이지 작성 (JSP)
- 학생의 이름, 국어, 영어, 수학 점수를 입력받는 JSP 페이지를 작성한다.
- form 태그를 사용하여 데이터를 Servlet으로 전송한다.
4. Servlet 작성 (저장소 역할)
- 하나의 Servlet을 생성한다.
- Servlet 내부에 ArrayList<Student>를 선언하여 학생 객체들을 저장한다.
- JSP로부터 전달받은 데이터를 이용해 Student 객체를 생성한다.
- 생성된 객체를 ArrayList에 저장한다.
5. 출력 페이지 작성 (JSP)
- Servlet에 저장된 Student 객체 목록을 JSP에서 불러온다.
- 저장된 학생들의 정보를 화면에 출력한다.
6. 스크립틀릿 사용
- JSP의 스크립틀릿 영역에서 Student 인스턴스를 사용하여 값을 출력한다.

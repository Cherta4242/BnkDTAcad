<h3>문제 2</h3>

파일 구성
- test0.jsp
- result.jsp

요구사항
- test0.jsp -> 이름을 입력할 수 있는 텍스트박스와 제출 버튼을 만든다.
- 입력한 이름을 result.jsp로 전송한다.
- result.jsp -> 전달받은 이름을 저장한다. (이전 데이터가 유지되어야 함)
- 저장된 이름들을 테이블로 출력한다.

테이블 형식: 번호 / 이름
- 이름을 한 번 등록할 때마다
- 번호가 1씩 증가해야 한다.

- 현재까지 등록된 총 인원 수를 출력한다.

<h3>Promise</h3>
<h4>사용: const promise = new Promise((resolve, reject)</h4>
<p>promise를 실행하고 실행가능이면 resolve를 호출, 실행 불가면 reject를 호출한다.</p>
<ul>
  <li>동기, 비동기</li>
  <li>상태</li>
  <li>then, chatch, finally</li>
</ul>

<h4>Promise의 상태</h4>
<ul>
  <li>pending</li>
  <li>fulfilled</li>
  <li>rejected</li>
</ul>

<h4>.then</h4>
<ul>
  <li>resulve가 호출되었을 때 실행되는 함수</li>
</ul>

<h4>.catch</h4>
<ul>
  <li>reject가 호출되었을 때 실행되늰 함수</li>
</ul>

<h4>.finally</h4>
<ul>
  <li>무조건 실행되는 함수</li>
</ul>
